Instalación

Para empezar la Instalación de Visual Studio Code, se debe descargar el ejecutable desde su página oficial. Existen dos tipos de instalaciones: User Installer y System Installer. La instalación User, no requiere permisos del administrador, las actualizaciones serán más sencillas. La instalación System se hará para todo el sistema. Se recomienda ésta última. Una vez descargado, se ejecuta y se acepta la licencia. Después se elige la ruta donde instalar el entorno de desarrollo y en el resto de ventanas sólo hay que pulsar el botón para seguir.

Complementos

Los complementos que permiten personalizar y mejorar la experiencia en Visual Studio mediante la incorporación de nuevas características o la integración de herramientas existentes.

Los más recomendados para JavaScript son:

  • Live server: permite una vista previa de su archivo HTML en un navegador.
  • JSCS Linting: Analiza el código JavaScript y revisa que se están siguiendo unos determinados estilos de programación a la hora de programar.
  • Javascript ES6: contiene fragmentos de código para JavaScript en la sintaxis de ES6 admite JavaScript y TypeScript.
  • Node Security Project: Esta extensión añade un nuevo comando, nsp, que lanza una verificación de tu código contra el inventario de vulnerabilidades de seguridad de Node Security Project.
  • Bookmarks: Permite marcar ciertas líneas en nuestro código y luego saltar rápidamente de unas a otras con unas teclas rápidas.

Funcionamiento

Para usar este entorno de desarrollo, primero de todo hay que crear una carpeta. Se hace clic en File/Open folder/ en el nuevo cuadro ubicamos y se selecciona con un clic la carpeta que se haya creado anteriormente y se hace clic en Select folder. Luego hay que iniciar el primer archivo HTML haciendo clic en el botón New file y agregamos el nombre y extensión respectivo: index.html.

Se empieza entonces a codificar nuestra primera web, para ello mientras se empieza a escribir la primera etiqueta htm se muestra un menú desplegable que indica la forma de iniciar la página web, con las teclas direccionales se selecciona html:5 y luego se pulsa enter, esto permitirá que los plugins instalados anteriormente nos ayuden a tener toda la estructura principal de nuestra página web.

Para guardar los cambios, hay que hacer clic en el menú File, clic en la opción Save. También se puede utilizar la combinación de teclas Ctrl + S.
