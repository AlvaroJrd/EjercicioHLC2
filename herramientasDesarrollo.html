Chrome DevTools

Chrome DevTools es un conjunto de herramientas de creación web y depuración integrado en Google Chrome. Usa DevTools para iterar y depurar tu sitio, y para crear un perfil de él.

Abrir Chrome DevTools

  • En el Menú de Chrome, selecciona More Tools > Developer Tools.
  • Haz clic con el botón secundario en un elemento de la página y selecciona Inspect.
  • Usa las combinaciones de teclas Ctrl+Mayúscula+I (Windows) o Cmd+Opción+I (Mac).

Firefox Developer Edition

Firefox Developer Edition es una versión de Firefox adaptada para desarrolladores, ofreciendo las últimas características y herramientas de desarrollo experimental. La actual edición de desarrollo (Firefox Developer Edition) incluye estos cambios en las herramientas de desarrollador:

  • Registro del servidor en la Consola Web
  • Busqueda rápida de la regla sobre escrita en la declaración del CSS
  • "Usar en Consola" en el item de menu contextual en el Inspector
  • Opción "estricto" para filtrado en las reglas de la vista
  • Entradas de red en la Consola ahora enlaza al Monitor de Red
  • Nueva barra lateral UI para WebIDE
